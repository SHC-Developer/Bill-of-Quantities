# 외관조사망도 도면별 물량표 추출 사이트

Excel 파일에서 도면별 물량표를 자동으로 추출하는 웹 애플리케이션입니다.

## 기능

- Excel 파일(.xlsx, .xls) 업로드
- Station별 손상내용 집계
  - 전체물량/전체개소 (보수 제외)
  - 신규물량/신규개소 (신규, 재손상, 재결함 포함)
- 결과를 웹에서 확인 및 Excel 파일로 다운로드

## 사용 방법

### 로컬에서 실행

1. 모든 파일을 같은 폴더에 저장합니다:
   - `index.html`
   - `script.js`
   - `style.css`

2. `index.html` 파일을 Chrome 브라우저에서 엽니다.
   - 파일을 더블클릭하거나
   - Chrome에서 `파일 > 파일 열기` 메뉴를 사용하거나
   - 주소창에 `file:///` 경로를 입력

3. Excel 파일을 업로드합니다:
   - "파일 선택" 버튼을 클릭하거나
   - 파일을 드래그 앤 드롭

4. 결과를 확인하고 필요시 Excel로 다운로드합니다.

### Excel 파일 형식

Excel 파일은 다음 형식을 따라야 합니다:

- **Sheet1**에 데이터가 있어야 합니다
- **A열**: Station 이름 (A~N 병합 셀)
- **C열**: 손상내용
- **G열**: 물량
- **I열**: 개소
- **N열**: 구분 (보수, 신규, 재손상, 재결함 등)

#### 예시 구조:

```
| A (Station) | ... | C (손상내용) | ... | G (물량) | ... | I (개소) | ... | N (구분) |
|-------------|-----|--------------|-----|----------|-----|----------|-----|----------|
| Station 1   | ... |              | ... |          | ... |          | ... |          |
|             | ... | 손상A        | ... | 10.5     | ... | 2        | ... | 신규     |
|             | ... | 손상B        | ... | 5.0      | ... | 1        | ... | 보수     |
| Station 2   | ... |              | ... |          | ... |          | ... |          |
|             | ... | 손상C        | ... | 8.0      | ... | 3        | ... | 재손상   |
```

## 기술 스택

- HTML5
- JavaScript (Vanilla JS)
- CSS3
- SheetJS (xlsx.js) - Excel 파일 읽기
- ExcelJS - Excel 파일 쓰기 및 스타일링

## 주의사항

- Chrome 브라우저에서 최적화되어 있습니다
- 모바일/태블릿 반응형은 지원하지 않습니다
- Excel 파일의 첫 번째 시트만 처리합니다
- Station 행은 A열에 값이 있고 C열이 비어있거나 데이터 열(G, I)이 비어있는 행으로 감지됩니다
